# Changelog

## [4.0.2] - 2026-01-15
### Fixed
- Merged PR #149: Consolidated duplicate `detect_distribution()` functions, fixing Ubuntu/Debian detection inconsistency (closes #148)
- Optimized suspend/resume hook to fix slow resume regression (Issue #147):
  - Single-pass loop instead of two separate loops
  - Reduced per-device sleep from 0.5s to 0.1s
  - Background RGB restore service restart (non-blocking)

## [4.0.1] - 2026-01-03
### Fixed
- Fixed empty `gz302-gaming.sh` and `gz302-hypervisor.sh` modules.
- **Gaming Module**:
    - Added CachyOS detection to use `cachyos-gaming-meta`.
    - Resolved Wine conflicts on Arch by using standard `wine`.
- Unified versioning across all scripts to use central `VERSION` file.

## [4.0.0] - 2026-01-03
### Changed
- **Major Refactor**: Decoupled Main Installer from Command Center Installer.
- `gz302-main.sh` now focuses strictly on hardware fixes (Kernel, GRUB, Audio, WiFi).
- `install-command-center.sh` now handles ALL user tools:
    - Power Management (`pwrcfg`) & Daemon (`power-profiles-daemon`).
    - Display Control (`rrcfg`) & Switcheroo.
    - RGB Control (`gz302-rgb`).
    - Tray Icon.
- Moved `power-profiles-daemon` and `asusctl` installation logic from Main Installer to Command Center Installer.
